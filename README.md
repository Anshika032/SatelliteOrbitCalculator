# 🚀 Space Orbit Simulation System

## 📌 Overview

This project calculates and visualizes satellite orbits using C++ and MATLAB.

## ⚙️ Files Included

* Orbit.cpp → Orbital calculations (C++)
* orbit.csv → Generated orbit data
* plot_orbit.m → Advanced 3D interactive simulation


## 🔥 Features

* Circular & Elliptical orbit calculation
* Multi-planet support
* 2D & 3D orbit visualization
* Real-time satellite animation
* Interactive controls (Play/Pause, Speed, Inclination)

## ▶️ How to Run

### Step 1: Run C++ Code

Compile and execute:
Orbit.cpp

### Step 2: Upload orbit.csv to MATLAB

### Step 3: Run:

orbit_simulator.m

## 🧠 Concepts Used

* Kepler’s Laws of Planetary Motion
* Orbital Mechanics
* Physics-based simulation
